### Technical Requirements

```yaml
Workstations (per participant):
  Hardware:
    - Modern laptop or desktop
    - 8GB RAM minimum (16GB recommended)
    - Dual monitors recommended
    - Webcam and microphone (for recordings)

  Software:
    - Modern web browser (Chrome, Firefox, Edge)
    - Text editor or IDE (VS Code recommended)
    - Terminal/command line access
    - Docker Desktop (for technical training)

  Access:
    - Administrative rights (for technical training)
    - VPN client (if accessing internal systems)

Network:
  Internet:
    - Minimum 50 Mbps symmetrical
    - Reliable, low-latency connection
    - No restrictive proxy/firewall blocking

  Firewall Rules:
    - HTTPS (443) outbound
    - WSS (WebSocket) support
    - SSH (22) for lab access

  WiFi:
    - Dedicated network recommended
    - WPA2/WPA3 security
    - Sufficient capacity for all participants

Lab Environment Access:
  - Cloud-based lab environment provided
  - Unique credentials per participant
  - VPN access for on-premises labs (if required)
```
